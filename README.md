# autocomplete-date package

> autocomplete date

<img src="https://raw.githubusercontent.com/akameco/autocomplete-date/master/screenshot.gif"/>

## Install

```
$ apm install autocomplete-date
```

## Usage
Provide snippet

### date (dd)
`YYYY-MM-DD`

### time (dt)
`HH:mm:ss`

### now
`YYYY-MM-DD HH:mm:ss`

See [Moment.js | Docs](http://momentjs.com/docs/#/displaying/).
